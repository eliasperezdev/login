# Template login - Sequelize - Express - Typescript

## Envinroment setup

1) Create database
2) Copy .env.example to .env and fill with database credentials.

To install dependencies, run

``` bash
npm install
```
3)Build:

``` bash
npm run build
```
4) Migrations:

``` bash
npm run migrate
```

5) Seeders:

``` bash
npm run seeders
```

## Start local server

``` bash
npm run dev
```
